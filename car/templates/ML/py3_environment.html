Python 环境安装

Python 是跨平台的，它可以运行在 Windows、Mac 和 Linux/Unix 系统上。在 Linux 上写 Python 程序，放到别的系统上也能够运行。 现在 Python 有两个版本，一个是2.x版，一个是3.x版，这两个版本是不兼容的，教程主要讲述 Python3 版本。

安装 Python3

Mac 安装: 如果你正在使用Mac，系统是OS X>=10.9，那么系统自带的Python版本是2.7。要安装最新的Python 3.8，有两个方法：

方法一：从Python官网下载Python 3.8的安装程序，下载后双击运行并安装.

方法二：如果安装了Homebrew，直接通过命令brew install python3安装即可。

Windows 安装: Python 官网下载地址

1、选择Python版本，点 download 进入下载页，这里我们以 Python 3.7.4 为例。

2、根据电脑系统选择64位或32位。这里我们选择64位安装版本，开始下载。下载完成后，执行安装包。

3、勾选Add Python 3.7 to PATH，然后点击Customize installation(自定义安装)。

4、点击Next。

5、点击红框旁边的Browse(浏览)按钮选择你想要的安装的位置，然后点击Install。

6、正在安装。

7、点击Close。

8、验证是否安装成功，Win+R键打开cmd窗口。

9、输入python，如下图所示即进入 Python 解释器，Python 安装成功；若未进入Python解释器，请检查每一步是否操作正确。


Linux 安装: Linux大部分系统自带 Python2.x 的版本，最常见的是 Python2.7 版本,自带的 Python 被很多系统程序依赖， 不要轻易删除， 否则会出问题。如需使用最新的Python3，可以编译安装源码包到独立目录， 不会影响到系统默认环境，python3和python2两个环境可以并存。 安装步骤如下：

1、安装依赖包

1）首先安装gcc编译器，gcc有些系统版本已经默认安装，通过 gcc --version 查看，没安装的先安装gcc。

yum -y install gcc

2）安装其它依赖包，（注：不要缺少，否则有可能安装python出错，python3.7.0以下的版本可不装 libffi-devel ）

yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel libffi-devel

2、源码下载地址: https://www.python.org/ftp/python/

根据自己需求选择对应的源码包，我下载的是 Python3.7.4 。

wget https://www.python.org/ftp/python/3.7.4/Python-3.7.4.tgz

3、解压Python-3.7.4.tgz

tar -zxvf Python-3.7.4.tgz

4、进入目录：

cd Python-3.7.4/

5、添加配置，用于保存 Python3 程序：

./configure --prefix=/usr/python3

5、添加配置，用于保存 Python3 程序：

./configure --prefix=/usr/python3

6、编译，编译安装

make && make install

安装完成没有提示错误便安装成功了

7、建立软连接

ln -s /usr/python3/bin/python3.7 /usr/bin/python3
ln -s /usr/python3/bin/pip3.7 /usr/bin/pip3

8、测试一下python3是否可以用。直接执行python3 命令，进入Python解释器则证明安装成功。
